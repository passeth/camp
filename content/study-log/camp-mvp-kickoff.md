# Camp MVP 킥오프

MVP는 공개 페이지, 멤버 인증, Markdown 콘텐츠 로더, 게시 요청 워크플로우를 갖춘 웹앱으로 정의했습니다.

## 구현 순서

- Next.js 앱을 만든다.
- 샘플 Markdown 콘텐츠를 렌더링한다.
- Supabase Auth와 승인제 권한을 붙인다.
- 웹 에디터에서 게시 요청을 생성한다.
